# q-read

A speed-reading web app inspired by RSVP (Rapid Serial Visual Presentation).

## Features
- Paste text and read word-by-word
- Adjustable WPM
- Pivot letter highlighting
- Play / Pause / Restart

## Tech Stack
- React (Vite)
- JavaScript
- CSS

## Getting Started

```bash
git clone https://github.com/RyanEAA/q-read.git
cd q-read
npm install
npm run dev
```
